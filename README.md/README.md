# Calorie Counter 🍽️

This is a simple web app that helps users track their daily calorie intake. It’s built with **HTML**, **CSS**, and **JavaScript**.

## 🔧 Features

- Add, edit, and delete food items
- See total calories consumed
- Lightweight and responsive
